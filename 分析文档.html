Week 13¶

  自人类迈入蒸汽时代以来迄今为止，获取能源以维持社会经济运转的方式大头始终是依赖燃烧——石油、煤、天然气，因此燃烧能源的消耗很多时候通过大气的碳排放即可推算得来。
  自千禧年以来，西方发达国家一直在号召碳中和计划，矛头直指发展中国家的碳排放巨大，对地球污染造成巨大影响，并且借此机会以通过限制碳排放的公约以间接完成限制他国经济发展水平的目的。

df2 = data1[(data1["Gas"]=="All GHG")&(data1["Sector"]=="Total including LUCF")]
country = df2["Country"].values[1:]
GHG = df2["2016"].values[1:]
G = pd.Series(GHG,index=country)
cmap = plt.cm.YlOrRd
colors = {s:cmap(np.sqrt((G[s]-G.min())/(G.max()-G.min())))[:3] for s in G.index}
plt.figure(figsize=(16,12))
mm = Basemap(projection="cyl",llcrnrx=-180,llcrnry=-90,urcrnrx=180,urcrnry=90)
mm.drawcoastlines(linewidth=0.2)
mm.drawcountries(linewidth=0.2)
mm.readshapefile("ne_50m_admin_0_countries","countries",drawbounds=False,default_encoding="ISO-8859-1")
count = []
for p in mm.countries_info:
    count.append(p["NAME"])
ax = plt.gca()
for c in G.index[:]:
    for name,seg in zip(count,mm.countries):
        if c in name:
            color = rgb2hex(colors[c]) # 将 RGB 色彩转为 HEX 色彩 
            poly = Polygon(seg, 
                   facecolor=color,
                   edgecolor=color) 
            ax.add_patch(poly)
        else:
            pass

  根据数据集数据情况，碳排放量最近的记录为2016年，其全球碳排放量热度图如上所示，颜色越深的区域碳排放量越大，其中能够明显的看出，中美两国属于碳排放量前二的国家，亚洲区域内的碳排放量最多，非洲、澳洲和南美洲颜色较浅，碳排放最少。

4、碳排放量与该国的经济状况是否有强相关性？¶

In [5]:
#获取各国国内生产宗旨（GDP_MD）
GDP_MD = []
for c in country:
    for p in mm.countries_info:
        if c in p["NAME"]:
            g = [c,p["GDP_MD"]]
            if g not in GDP_MD:
                GDP_MD.append(g)
GDP_MD = pd.DataFrame(GDP_MD,columns=["Country","GDP_MD"])
GDP_MD = GDP_MD.sort_values("GDP_MD",ascending=False)
GDP_MD = GDP_MD.drop_duplicates("Country",keep="first")
df4 = df2[df2["Country"].isin(GDP_MD["Country"])]
GDP_MD.index = GDP_MD["Country"]
df4.index = df4["Country"]
df4 = pd.concat([GDP_MD,df4],axis=1)[["GDP_MD","2016"]]
#计算碳排放量与该国的GDP的相关性
kendall = df4.corr("kendall").values[0][1]
spearman = df4.corr("spearman").values[0][1]
pearson = df4.corr("pearson").values[0][1]
print("2016年各国碳排放量与GDP的相关性系数：")
print("kendall相关性系数：%s"%kendall)
print("spearman相关性系数：%s"%spearman)
print("pearson相关性系数：%s"%pearson)

2016年各国碳排放量与GDP的相关性系数：
